How matching actually works

No black box. Here's the real algorithm.

Hard filters first

Some things aren't preferences — they're requirements. These eliminate a pair entirely, no matter how good the rest looks:

  • Mutual gender preference (checked both directions)
  • Age range (both directions)
  • Location
  • Stated dealbreakers — a "no smokers" is a wall, not a weight

Then weighted compatibility

  • 30% Relationship intent alignment
  • 25% Shared interests
  • 25% Shared values
  • 20% Personality complementarity

Two people both looking for something long-term, with overlapping values and a compatible sense of humor, beat a hundred mutual right-swipes.

Privacy is structural

  • Phone numbers are SHA-256 hashed — never stored in plaintext
  • Reverse lookups use AES-256-GCM encryption
  • The database accepts zero client connections — server only
  • Video rooms are anonymous, time-limited, auto-deleted
  • Your name is shared exactly once: when you both consent
