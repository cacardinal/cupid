   `Inbound SMS {"from":"+1314****00","bodyLength":37}` (phone masking works).

### What this run proves / does not prove

- **Proves:** webhook routing + form parsing (Twilio `From`/`Body`), user creation,
  phone hashing + encrypted reverse-mapping, conversation persistence and
  multi-turn continuity, Firestore security rules enforcement, live outbound
  wiring to both Anthropic and Twilio APIs (each rejected the stub creds at auth,
  i.e., requests are correctly formed and dispatched).
- **Does not prove:** a real Claude-generated reply, assistant-turn persistence,
  profile extraction/merge, or an actual outbound SMS (all gated on real keys).
